A flexible REST API framework for WordPress, allowing access to any database table with API key management and analytics.

== Description ==

TechSpace REST Framework is a powerful plugin that extends WordPress's REST API capabilities. It allows you to:

* Generate API keys for secure access
* Access data from any WordPress database table
* Track API usage with built-in analytics
* Customize access permissions for different users

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/techspace-rest-framework` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->TechSpace API screen to configure the plugin

== Frequently Asked Questions ==

= How do I generate an API key? =

Navigate to the TechSpace API settings in your WordPress admin panel and click on "Generate New API Key".

= Can I restrict access to certain tables? =

Yes, you can configure table access in the plugin settings.
